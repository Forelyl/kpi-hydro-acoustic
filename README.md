## **Frontend**

### need to have
npm - https://nodejs.org/

### to install
```sh
npm install
```

### to run
```sh
npm run dev
```

## **Backend**
`in the Backend folder`

### need to have
poetry - https://python-poetry.org/

### to install
```sh
poetry install
```

### to run
open poetry env
```sh
poetry shell
```

run custom start command
```sh
poetry run run_dev
```

### to test
- use Bruno (https://www.usebruno.com/)
- in Bruno open folder "Backend/test/REST client"
